How to change the offer / affiliate links

When an offer caps or you switch networks, you change where every "Buy" button sends traffic. It's one file, one edit, live in ~1 minute — no coding, no rebuild.

⚡
TL;DR: edit the OFFERS list in assets/config.js → paste the new offer url (the Influx tracking link) for the geo(s) → commit. That's it.
✏️ Open config.js to edit →

Steps

1

Open the one file that controls the links

Go to github.com/paulblack6522/trycoolizi → assets/config.js (the ✏️ button opens it ready to edit).

You need write access to the repo — ask the account owner to add you as a collaborator once.

2

Find the OFFERS block

Near the top you'll see a big banner: OFFER LINKS — EDIT HERE…. Right under it is the list of every geo:

var OFFERS = {
  //  geo :  { url: "<Influx tracking link>",            s1: "<tracking tag>" }
  "en": { url: "https://www.abjdoi4kjd.com/27DJQ2C/D6NGN16/", s1: "try-uk" },  // UK / Ireland
  "de": { url: "https://www.abjdoi4kjd.com/27DJQ2C/D6JB1RC/", s1: "try-de" },  // DE / AT / CH
  "fr": { url: "https://www.abjdoi4kjd.com/27DJQ2C/D6M3R8K/", s1: "try-fr" },  // FR / BE
  "it": { url: "https://www.abjdoi4kjd.com/27DJQ2C/D726T5F/", s1: "try-it" },  // Italy
  "es": { url: "https://www.abjdoi4kjd.com/27DJQ2C/D73KPW2/", s1: "try-es" },  // Spain
  "nl": { url: "https://www.abjdoi4kjd.com/27DJQ2C/D6W26XL/", s1: "try-nl" },  // Netherlands
  "pt": { url: "https://www.abjdoi4kjd.com/27DJQ2C/D6ZRZDS/", s1: "try-pt" },  // Portugal
  "el": { url: "https://www.abjdoi4kjd.com/27DJQ2C/D6XF3N7/", s1: "try-gr" }   // Greece
};
3

Change the url for the geo(s) you're switching

In Influx, open the offer and copy its tracking link. Paste that whole link into url for the right geo. Example: a new DE offer → its Influx link is https://www.abjdoi4kjd.com/27DJQ2C/XXXX/ → set "de": { url: "https://www.abjdoi4kjd.com/27DJQ2C/XXXX/", s1: "try-de" }.

⚠️ Leave s1 alone unless the network explicitly tells you to change it — it's our per-geo tracking tag, and the dashboard reads it.

4

Commit the change

Scroll down, write a short message like switch DE offer to 9540, and click the green Commit changes button.

5

Done — it's live

The new link is live across all pages in ~1 minute (no rebuild). To double-check: open the geo's page (e.g. trycoolizi.com/de/), click a Buy button, and confirm it lands on the right offer.
